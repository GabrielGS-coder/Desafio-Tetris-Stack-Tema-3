## Descrição do Projeto
Este projeto implementa uma simulação do empilhamento de peças do jogo Tetris utilizando o conceito de Pilha (Stack), estudado na disciplina de Estrutura de Dados.

Cada peça é empilhada no topo da pilha e pode ser removida seguindo o princípio LIFO (Last In, First Out — o último que entra é o primeiro que sai).

O programa roda no terminal e apresenta um menu interativo para manipulação das peças.

---

## Funcionalidades
- Empilhar peça: adiciona uma nova peça ao topo da pilha.  
- Desempilhar peça: remove a última peça inserida.  
- Mostrar pilha: exibe todas as peças empilhadas da base ao topo.  
- Sair: encerra o programa.
